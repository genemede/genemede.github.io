Name of the principal investigator.

Name of the institution.

Name of the department.

Location of the institute and department.

Contact telephone number.

Contact email.
contact_website
Contact website.
contact_address
Contact address.
project_start_date
Start date of the project.
project_end_date
End date of the project
project_lead
Name of the project leader / person responsible for the project.
members
Members of the project
project_manager
Name of the project manager.
data_manager
Name of the data manager.
project_digital_tools
List of digital tools used in the project. e.g. slack, slab, keeper, trello
project_legacy_management
Documentation that describes the legacy of the project and the data. This may include documents describing policies on how how the project and all related output will be maintained.
